# 🎓 SkillHub Management System

A web-based SkillHub Management System developed to manage courses, students, and enrollments efficiently. This system provides an easy-to-use interface for administrators to organize training programs and manage student information.

## 🚀 Features

- User Authentication (Login & Registration)
- Student Management (Add, Update, Delete, Search)
- Course Management
- Enrollment Management
- Dashboard for Administrators
- Responsive User Interface

## 🛠️ Technologies Used

- Java
- Java EE
- JSP
- Servlet
- HTML
- CSS
- JavaScript
- Apache Tomcat
